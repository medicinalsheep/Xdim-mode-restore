# Xdim-mode-restore
Restores the softer 'Dim' dark theme on x.com with a slider bar instead of pure black 'Lights Out'.

How to install & test (developer mode)Create a folder, put all files above inside.
Open Chrome → chrome://extensions/
Enable "Developer mode" (top right)
Click "Load unpacked" → select your folder
Go to x.com, switch to dark mode (Settings → Premium → Display → Lights Out if needed)
The extension should auto-apply (default on). Use the toolbar popup
